# Tutorials

Welcome to the tutorial collection!  

Here you'll find a series of hands-on guides that let you explore, analyze, and visualize drought (in conjuction with exposure and vulnerability) data — from quick introductions to more detailed workflows.

Under **Storylines**, you'll find risk-oriented tutorials:
- [The return period of extreme precipitation](tutorials/return_period.ipynb)
- [Extreme precipitation in the past, present, and future](tutorials/changing_hazard.ipynb)
- [People exposed to an increasing frequency of extreme precipitation events](tutorials/exposed_people_extreme_precip.ipynb)

Under **Explore Exposure Data**, you'll find tutorials that read and visualise exposure data sets: 
- [Understanding Population Data](tutorials/population.ipynb)

💡 **Tip:** Many of these tutorials can be run with data already stored in the GitHub repository. If you'd like to create similar datasets for a different region, check out the [how-to guides](how-to-guides.md)!

**The jupyter notebooks in these tutorials can be run on free online platforms, such as Binder, Kaggle and Colab, or they can be accessed from GitHub. The links to run this notebook in these environments are provided here, but please note they are not supported by ECMWF.**
